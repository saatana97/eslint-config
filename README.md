> TypeScript 和 Vue 项目的 ESLint 配置文件

## 💎 使用方式

```cmd
npm install --save-dev @xwink/eslint-config
pnpm add --save-dev @xwink/eslint-config
```

## 📖 快速入门

```typescript
// .eslintrc
{
    "extends": "@xwink"
}
// in vue projects
{
    "extends": "@xwink/eslint-config/vue"
}
```
